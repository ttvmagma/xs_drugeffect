Das Script kann deinen Items einen Effekt verleihen wie in den Beispielen gezeigt

----------------
Speed:
- Schneller Rennen
- 50% Weste
----------------
Kokakin:
- 25% Weste
----------------
Joint:
- 100% Weste
----------------
Du kannst beliebig viele Items angeben
